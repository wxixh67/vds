⚠️ 安全警告

  • 仅在自己的设备上使用自己的Cookie
  • 切勿使用他人的Cookie，这属于非法访问
  • Cookie包含敏感信息，注意保护隐私
  • 此方法可能违反Facebook服务条款

🍪 Facebook Cookie 注入器

📌 使用说明：
1. 确保当前正在访问 m.facebook.com
2. 粘贴完整的Cookie字符串
3. 点击"注入Cookie"按钮
4. 页面将自动刷新并登录

如何获取Cookie？

  1. 在已登录的Facebook页面按F12打开开发者工具
  2. 切换到"Console"（控制台）标签
  3. 输入：document.cookie 并回车
  4. 复制显示的Cookie字符串
  5. 粘贴到上方文本框中
